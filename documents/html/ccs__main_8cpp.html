Main function for the ARIAC competition node.

Initializes ROS2, creates an instance of the CompetitionARIAC node, and runs the node using a multi-threaded executor. This setup allows the node to manage various functionalities required for the ARIAC competition, such as handling orders, controlling robots, and processing sensor inputs.
