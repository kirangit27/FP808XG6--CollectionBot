\doxysubsection{Detailed Description}
Class to manage part properties and configurations in the A\+R\+I\+AC competition. 

This class includes constants and mappings related to parts used in the A\+R\+I\+AC competition. It contains details like part types, colors, heights, and specific configurations for handling parts with the robotic system.
