\doxysubsection{Detailed Description}
This class manages the A\+R\+I\+AC competition workflow and robot interactions. 

It subscribes to various topics to receive competition state updates, orders, and sensor data. It also provides functionalities to control a robot for tasks like picking, placing, and moving parts.
